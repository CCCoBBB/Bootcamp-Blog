I was stuck on a simple problem where I had written a line of code that was in the wrong place, I felt very stuck and so I used a
few simple problem solving techniques that allowed me to progress. I used googling, console logging and asking peers

elegantly solved a problem

I solved a problem during the javascript introduction's function section. It was the last question of this section and I was
mulling over this question for hours. I was to take a function with a parameter to console log it with the parameter duplicated
with a space between each string, so the return would be the same value twice. After a very long time trying to bash the answer
out of my keyboard, I just read the error messages and rewrote the code several times until the error messages went away.
