    The DOM

    The DOM or Document object model, is a tree like structure that represents content on an HTML website.
    The DOM can be used to manipulate elements of a website. The tree can be visualized as a familty where the root
    would be the window/document itself. As the tree grows, the branches would represent big chunks of the document
    such as a header or body or the footer. The smaller branches afterwards would be elements within the chunks
    such as buttons, text or images.
